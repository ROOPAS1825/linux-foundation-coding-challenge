# Linux Foundation Coding Challenge

## Tower of Hanoi using Python

This project demonstrates the Tower of Hanoi problem using recursion in Python.

### Concepts Used
- Recursion
- Function calls
- Problem decomposition

### How to Run

```bash
python tower_of_hanoi.py
```

### Sample Output

```text
Move disk 1 from A -> C
Move disk 2 from A -> B
Move disk 1 from C -> B
```

This project was created as part of the Linux Foundation mentorship coding challenge.
